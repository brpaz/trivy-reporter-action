# Agents

This project consissts of a GitHub Action that is responsible to parse the JSON output file from a Trivy vulnerability scan and create or update a GitHub Issue with the security report.

## Technologies Used

- **GitHub Actions**: The project is implemented as a GitHub Action, allowing it to be easily integrated into CI/CD pipelines on GitHub.
- **JavaScript/Node.js**: The action is built using JavaScript and runs on Node.js, leveraging its ecosystem for handling file operations and GitHub API interactions.
- **GitHub API**: The action interacts with the GitHub API to create or update using Octokit, a GitHub API client for JavaScript.
- **Trivy**: The action processes the JSON output generated by Trivy, a popular open-source vulnerability scanner for containers and other artifacts.
- **Vitest**: The project uses Vitest for unit testing, ensuring the reliability and correctness of the action's functionality.
